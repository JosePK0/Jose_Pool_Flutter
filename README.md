## 🚀 **Comenzando**

Si estás aquí, es porque quieres algo épico. Aquí te dejo los pasos para que empieces:

### **1. Clona el repositorio**
```bash
git clone https://github.com/JosePK0/cook_book.git
cd cook_book
```

### **2. Instala las dependencias**
```bash
flutter pub get
```

### **3. Ejecuta la app**
```bash
flutter run
```
